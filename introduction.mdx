## What is SameVibe?

SameVibe is a full-stack micro-blogging platform built with modern web technologies. It allows users to share their stories, anecdotes, and experiences with a vibrant community. With a sleek, responsive interface and powerful features, SameVibe makes it easy to connect and engage with others who share your vibe.

## Key Features

<AccordionGroup>
  <Accordion title="🔐 Secure Authentication" icon="lock">
    User registration and login with bcrypt password encryption. Session tokens are stored securely with automatic invalidation on new logins.
  </Accordion>
  
  <Accordion title="✍️ Post Management" icon="pen">
    Create, read, and explore posts from the community. Each post includes rich content support with title and body text.
  </Accordion>
  
  <Accordion title="🛡️ Protected Routes" icon="shield">
    Custom Next.js middleware ensures authenticated-only access to protected pages, with automatic redirects for unauthenticated users.
  </Accordion>
  
  <Accordion title="🎨 Modern UI" icon="palette">
    Built with Tailwind CSS, featuring glassmorphism effects, dark mode support, and a fully responsive design that works on all devices.
  </Accordion>
  
  <Accordion title="⚡ Performance" icon="bolt">
    Powered by Next.js 16 with App Router, React 19, and optimized database queries through Prisma ORM.
  </Accordion>
</AccordionGroup>

## Tech Stack

SameVibe is built with a cutting-edge technology stack designed for performance, developer experience, and scalability:

- **Framework**: [Next.js 16](https://nextjs.org/) with App Router
- **UI Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Database**: PostgreSQL with [Prisma ORM](https://www.prisma.io/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) primitives
- **Icons**: [Lucide React](https://lucide.dev/)
- **Authentication**: bcryptjs for password hashing
